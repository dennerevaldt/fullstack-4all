## Frontend Ekki

- Configurar base url na pasta `src/services/api` para a url:porta da api, por padrão é `http://localhost:3333`;

- Instalar dependências:

  - `NPM` npm install
  - `YARN` yarn;

- Iniciar:

  - `NPM` npm start;
  - `YARN` yarn start;

- Gerar build:

  - `NPM` npm run build;
  - `YARN` yarn build;
